# Directus monorepo
## Install

Run the following command:

```sh
pnpm install
```

Add following records to `/etc/hosts`
```sh
127.0.0.1       api.example.com
127.0.0.1       example.com
```

## Run

Run the following command:

```sh
pnpm run dev
```
